# 封面设计规范

> 从 SKILL.md 提取的封面设计详细规范。

## 封面组件

封面图由 `CoverImage.tsx` 组件生成（场景底图 + 暗角 + 文字叠加），同时作为：
1. 视频首帧（2秒静帧） — 各平台自动提取
2. 独立 PNG 导出 — 用于支持上传封面的平台（如 B站 API `cover_up`）

## 封面文案结构（2 层 + 描述）

封面标题不区分主副标题，只用一个标题即可。帖子标题用于各平台发布时的标题栏。

| 层级 | 要求 | 字号 | 示例 |
|------|------|------|------|
| **标题** | **2-6 个字**，最大最醒目，封面核心文字 | 120-160px, bold | 人类最后的发明 / 抄写员 |
| **帖子标题** | **一句话概括**，用于各平台标题栏（不在封面上显示，或作为小字副文案） | 36-48px | 人类造了个神, 神给人类造了座笼子 |
| **内容介绍** | 150-250 字，用于视频描述栏（不在封面上） | — | 2024年，陈远是一名AI公司的算法工程师... |

## 视觉布局（从上到下）

1. 补充标题（最小字号，金色半透明 `rgba(212,165,116,0.75)`）
2. 上分割线（120px 渐变金线）
3. 主标题（白色 160px，带金色光晕阴影）
4. 下分割线
5. 副标题（暖白色 48px）

## 底图选择

从已有场景图中选一张氛围合适的（如校园、人物远景），加暗角蒙版（`radial-gradient` 从 45% 到 80% 透明度），确保文字可读。

## 安全区域

各平台从首帧自动裁切不同比例，核心内容（标题文字、人物面部）必须放在 **画面中心 60%** 区域内：

```
┌──────────────────────────────┐ 16:9 原始
│                              │
│    ┌──────────────────┐      │
│    │  ┌────────────┐  │      │
│    │  │ 安全区域   │  │      │  ← 标题/人脸放这里
│    │  │  (中心60%) │  │      │
│    │  └────────────┘  │      │
│    │    4:3 裁切区域   │      │
│    └──────────────────┘      │
│                              │
└──────────────────────────────┘
```

| 平台 | 裁切比例 | 从 16:9 首帧裁切方式 |
|------|---------|-------------------|
| B站首页推荐 | 4:3 | 中心裁切，左右各切 ~11% |
| B站个人空间 | 16:9 | 原始比例，不裁切 |
| 视频号个人主页 | 3:4 | 中心裁切，左右各切 ~33% |
| 视频号分享卡片 | 4:3 | 中心裁切，左右各切 ~11% |
| 小红书 | 3:4 | 中心裁切，左右各切 ~33% |

## 输出三个比例

- `CoverImage` — 1920×1080 (16:9)，用作视频首帧
- `CoverImage43` — 1440×1080 (4:3)，B站首页/视频号分享卡片
- `CoverImage34` — 1080×1440 (3:4)，小红书/视频号个人主页

封面文案在 Phase 1 脚本拆分阶段一并产出。
