# 时间轴

> 生成视频章节导航条，支持B站/YouTube格式

## 功能

- 自动识别话题切换点
- 生成章节标题（2-4字）
- 输出B站/YouTube时间轴格式
- 烧录底部章节导航条到视频

## 使用场景

- B站投稿，方便观众跳转
- 长视频章节导航
- 内容结构预览

## 效果预览

**视频底部导航条**：

```
┌─────────────────────────────────────────────────┐
│              [视频画面内容]                      │
├─────────────────────────────────────────────────┤
│  开场白 │ 核心演示 │ 问题解答 │ 进阶技巧 │ 总结  │
│           ↑高亮                                 │
└─────────────────────────────────────────────────┘
```

**B站简介格式**：

```
00:00 开场白
01:17 核心演示
03:47 问题解答
06:00 总结
```
