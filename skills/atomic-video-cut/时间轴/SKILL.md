# 时间轴

> 自动识别话题切换 → 生成视频章节 → 烧录底部导航条

## 快速使用

```
用户: 帮我生成这个视频的时间轴
用户: 给视频加章节导航
用户: 做一个B站时间轴
```

## 适用场景

| 场景 | 说明 |
|------|------|
| B站投稿 | 生成评论区/简介时间轴 + 视频底部导航条 |
| 长视频 | 教程/课程/演讲，方便定位章节 |
| 内容回顾 | 快速了解视频结构 |

## 流程

```
1. 读取转录数据（transcript.json 或 SRT）
    ↓
2. 滑动窗口分析，识别话题切换点
    ↓
3. 生成章节标题（2-4字）
    ↓
4. 合并短段落（<30s）
    ↓
5. 输出时间轴.md + B站格式文本
    ↓
【用户审核修改】
    ↓
6. 烧录章节导航条到视频（FFmpeg）
    ↓
7. 输出带导航条的视频
```

### 优先级

| 优先级 | 数据源 | 说明 |
|--------|--------|------|
| 1 | transcript.json | 有精确字符级时间戳 |
| 2 | SRT 字幕文件 | 有句子级时间戳 |

### 查找顺序

```
1. 当前目录 *_transcript.json
2. 当前目录 *.srt
3. 精简功能输出的内容结构分析
```

## 二、话题切换识别

### 切换信号

| 信号 | 权重 | 示例 |
|------|------|------|
| 转折词开头 | 0.4 | "接下来"、"下面"、"另外" |
| 结构标记 | 0.5 | "第一"、"首先"、"最后" |
| 长静音后 | 0.3 | 静音 >= 2秒后的新内容 |
| 疑问句后 | 0.2 | 问答式切换 |
| 主题词变化 | 0.3 | 前后30秒名词重合度低 |

### 判断规则

- 信号权重累计 >= 0.5 → 标记为切换点
- 切换点之间最小间隔 30 秒
- 开头和结尾必是切换点

---

## 三、章节标题规范

### B站进度条标题（首选）

- 字数：**2-4字**
- 风格：名词短语或动宾结构
- 示例：`开场白`、`核心演示`、`问题解答`、`总结`

### 详细版标题

- 字数：5-8字
- 风格：更完整的描述
- 示例：`功能概述介绍`、`常见问题解答`

### 格式规范

| 规范 | 正确 | 错误 |
|------|------|------|
| 动宾结构 | "安装依赖" | "依赖" |
| 无代词 | "功能概述" | "介绍它" |
| 无语气词 | "问题解答" | "来看看常见问题" |
| 无标点 | "核心演示" | "核心演示！" |

---

## 四、段落合并规则

### 最小段落时长

- 普通段落：30 秒
- 开场段落：可以 < 30 秒
- 结尾段落：可以 < 30 秒

### 合并策略

```
短段落（<30s）
    ↓
计算与前/后段的语义相似度
    ↓
合并到更相似的段落
    ↓
保留合并前的摘要（如果更准确）
```

## 五、输出格式

### 文件命名

```

01-xxx_时间轴_B站.txt  # B站格式（可复制）
01-xxx-导航.mp4       # 带导航条的视频
```

### 时间轴.md 内容

```markdown
# 视频时间轴

**视频**: [文件名]
**总时长**: [MM:SS]
**章节数**: [N]

---

| 时间 | B站标题 | 详细标题 | 时长 |
|------|---------|----------|------|
| 00:00 | 开场白 | 自我介绍和背景 | 1:17 |
| 01:17 | 核心演示 | 核心功能操作演示 | 2:30 |
| ... | ... | ... | ... |

---

## B站格式（可复制到简介）

00:00 开场白
01:17 核心演示
03:47 问题解答
06:00 总结

## YouTube 格式

Chapters:
0:00 开场白
1:17 核心功能操作演示
3:47 常见问题解答
6:00 总结与展望
```

## 六、烧录底部导航条

### 效果预览

```
┌─────────────────────────────────────────────────┐
│                                                 │
│              [视频画面内容]                      │
│                                                 │
│           这是核心功能的演示...                  │  ← 字幕
├─────────────────────────────────────────────────┤
│  开场白 │ 核心演示 │ 问题解答 │ 进阶技巧 │ 总结  │  ← 导航条
│           ↑高亮                                 │
└─────────────────────────────────────────────────┘
```

### 样式说明

- 视频底部固定显示一行章节标签
- 标签横向排列，用竖线分隔
- **当前章节高亮**（黄色），其他章节灰色
- 随播放时间自动切换高亮

### FFmpeg 实现

```bash
# 为每个章节生成两套 drawtext
ffmpeg -i input.mp4 \
  -vf "
    # 底部半透明黑色背景条
    drawbox=x=0:y=ih-40:w=iw:h=40:color=black@0.7:t=fill,

    # 章节1：开场白 (0-77秒高亮)
    drawtext=text='开场白':fontsize=20:fontcolor=yellow:x=50:y=ih-30:enable='between(t,0,77)',
    drawtext=text='开场白':fontsize=20:fontcolor=gray:x=50:y=ih-30:enable='not(between(t,0,77))',

    # 章节2：核心演示 (77-150秒高亮)
    drawtext=text='核心演示':fontsize=20:fontcolor=yellow:x=150:y=ih-30:enable='between(t,77,150)',
    drawtext=text='核心演示':fontsize=20:fontcolor=gray:x=150:y=ih-30:enable='not(between(t,77,150))'
  " \
  -c:a copy output.mp4
```

### 样式配置

| 参数 | 默认值 | 说明 |
|------|--------|------|
| 背景色 | black@0.7 | 半透明黑底 |
| 高亮色 | yellow | 当前章节 |
| 普通色 | gray | 其他章节 |
| 字体大小 | 20 | 根据分辨率调整 |
| 导航条高度 | 40px | 底部条高度 |

### 用户可自定义

```
用户: 高亮用白色
用户: 字体大一点
用户: 导航条放顶部
```
